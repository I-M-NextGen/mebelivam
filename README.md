**[MebeliVam](https://www.mebelivam.com/)** is a high-performance landing page designed for **Mebeli VaM Ltd.**, a company specializing in bespoke furniture. The project serves as a digital bridge between potential clients and expert craftsmen, focusing on conversion through professional consultations.

## âœ¨ Features

- **Conversion Optimized:** Includes an interactive 30-minute consultation booking flow.
- **Modern UI/UX:** Built with a clean, professional aesthetic using `shadcn/ui`.
- **Responsive Architecture:** Seamless experience across mobile, tablet, and desktop.
- **Dynamic Content:** Sections for project workflows, FAQs, and service descriptions.
- **Performance Focused:** High Core Web Vitals scores thanks to Next.js 15 optimization.
